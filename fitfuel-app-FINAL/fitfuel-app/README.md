## 📱 About

MyFitness Meals is a comprehensive fitness meal delivery application that connects users with professional chefs for personalized, healthy meal planning.

---

## ✨ Features

- 🔐 Phone + OTP Authentication
- 🏠 Home Dashboard with Stats
- 🍽️ Meal Browser (Search & Filter)
- 📊 Progress Tracking
- 👤 Profile Management
- 💳 Subscription Plans

---

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Scan QR code with Expo Go app
```

**See [SETUP.md](SETUP.md) for detailed instructions.**
